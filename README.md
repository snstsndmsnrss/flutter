# Знакомство с Flutter

## Описание проекта
Создание базового пользовательского интерфейса на Flutter с использованием основных виджетов: `Container`, `Row`, `Column`, `Text`, `Scaffold`, `FloatingActionButton`, `CircleAvatar`.

## Цель работы
1. Освоить начальную настройку среды разработки для Flutter
2. Познакомиться с процессом создания, запуска и публикации Flutter-проекта
3. Изучить базовые виджеты Flutter и их свойства
4. Научиться создавать простой пользовательский интерфейс
